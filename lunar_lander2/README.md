# lunar_lander2

Projecte Lunar Lander
===

Descripció del projecte
---

És un diseny web del Lunar Lander original seguint el procediment del client. El seu document en format PDF es situa a la carpeta [contribucions client](../master/contribucions client)
**Client:** *Deniz Uysal Canaves*


Instruccions seguides
---
&#10004; **L'estructura de directoris: A /img/ posa les imatges, a /css/ posa els .css, al directori arrel posa tots els documents .html**

&#10060; **Les versions dels documents HTML per diferents disposituis han de contenir el mateix codi HTML excepte l'enllaç als CSS corresponent.**
*Només he inclòs una versió HTML per a tots els dispositius ja que usa un diseny responsive*

&#10004; **No usar JS! (encara no!!)**
*JavaScript s'inclourà més endavant.*

&#10004; **Visualització i aspecte (tant vertical com horitzontal)**

&#10004; **Documentació adequada al README.md**

&#10060; **Multidispositiu (crear un CSS per cada dispositiu, no usar ni media query ni js, encara no!!!)**

&#10004; **Respectar, en essència, la idea original del document**


&#10004; **Responsive**
&#10004; **Versions diferents del projecte, amb url de github diferents (branques) amb versions de CSS, HTML.**
&#10060; **Ús adequat de sprites.**

Modificacions
---
S'ha seguit el model del client sense pràcticament modificacions.
+ S'ha creat un diseny per a la pàgina d'informació.
+ Les imatges de la nau encesa/apagada i rompuda no s'acabaràn de emplear fins que no s'hagi implementat el JS.
